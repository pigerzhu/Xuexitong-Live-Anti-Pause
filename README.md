# Xuexitong-Live-Anti-Pause

## 📖 项目简介

学习通直播增强工具 - 解除学习通直播切换窗口时的自动暂停限制，支持后台播放，提升多任务学习体验。


## 📥 安装步骤

### 第一步：安装 Tampermonkey

根据你的浏览器选择对应的扩展：

- **Chrome / Edge**：在扩展商店搜索 [Tampermonkey](https://www.tampermonkey.net/) 并安装
- **Firefox**：在扩展商店搜索 Tampermonkey 或 Greasemonkey 并安装

### 第二步：导入脚本

1. **下载脚本文件**
   - 点击本项目的 `anti-pause-1.0.user.js` 文件
   - 点击右上角 `Download` 按钮下载

2. **导入到 Tampermonkey**
   - 点击浏览器右上角的 Tampermonkey 图标
   - 选择「**管理面板**」或「**Dashboard**」
   - 点击「**实用工具**」或「**Utilities**」标签
   - 在「**导入**」区域，点击「**选择文件**」
   - 选择刚才下载的 `.user.js` 文件
   - 点击「**安装**」确认

3. **确认脚本已启用**
   - 在 Tampermonkey 管理面板中查看脚本状态
   - 确保脚本开关为「开启」状态

4. 现在可以自由切换窗口，视频不会暂停 🎉
